pyathena
========

**pyathena** is a python analysis tool for simulation results from Athena4.2

Basic Usage
-----------

.. code-block:: python

    import pyathena as pa

    ds = pa.LoadSim()

A more complete example is available in the :ref:`pyathena-example1` tutorial.
